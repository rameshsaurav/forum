Mozilla to extend its 'total cookie protection' feature to Firefox 89 browser

Firefox says its Total Cookie Protection is a set of "privacy improvements" based on a principle that a user's browser should not allow sharing of cookies between websites. It's now enforced in Firefox Private Browsing windows and helps by creating a separate cookie jar for every website that the user visits.
